# Hola Buñuelos

Hola a todos,

En este blog, quiero hablar sobre las diferentes maneras en que se pueden utilizar las funciones sobre vectores en JavaScript.

La primera manera de utilizar funciones sobre vectores en JavaScript es mediante el uso de la función map(). Esta función permite aplicar una función a cada elemento de un vector y devolver un nuevo vector con los resultados. Por ejemplo, podemos utilizar map() para multiplicar cada elemento de un vector por 2:

```javascript
const vector = [1, 2, 3, 4, 5];

// Usamos la función map() para multiplicar cada elemento del vector por 2
const result = vector.map(x => x * 2);

// El resultado será un nuevo vector con los valores 2, 4, 6, 8, 10
console.log(result);
}
```
